## 🇪🇸 Versión en Español

### 📝 Descripción

**MetroidVaniaDark** es un videojuego de acción y plataformas 2D desarrollado en **Unity**. El jugador controla a un ninja que debe abrirse paso a través de niveles infestados de zombis, utilizando habilidades de combate ágiles y exploración de escenarios con una estética oscura y atmosférica.

### ✨ Características Principales

- ⚔️ **Mecánicas de Combate** — Sistema de ataque cuerpo a cuerpo con katana y gestión de vida del jugador.
- 🏃 **Movimiento Ágil** — Salto y detección de colisiones optimizada para un gameplay fluido y responsivo.
- 🧟 **Enemigos con IA** — Zombis con comportamiento de persecución, detección de rango y ataque.
- 🌑 **Estética Dark** — Ambientación oscura lograda mediante Sprites 2D y capas de Parallax scrolling.

## 🇬🇧 English Version

### 📝 Description

**MetroidVaniaDark** is a 2D action-platformer developed in **Unity**. Players control a ninja who must fight through zombie-infested levels using agile combat mechanics and environment exploration, wrapped in a dark and atmospheric Metroidvania aesthetic.

---

### 📸 Screenshots

> *(Add screenshots or gameplay GIFs here)*

---

### ✨ Key Features

- ⚔️ **Combat Mechanics** — Melee attack system (katana) with player health management.
- 🏃 **Agile Movement** — Jumping and optimized collision detection for fluid, responsive gameplay.
- 🧟 **AI Enemies** — Zombies with range detection, chasing, and attack behaviors.
- 🌑 **Dark Aesthetics** — Dark atmosphere achieved through 2D Sprites and Parallax scrolling layers.

---

### 🎮 Controls

| Action | Input |
|:---|:---|
| Move | `A` / `D` or `←` / `→` |
| Jump | `Space` |
| Attack | `J` or `Left Click` |
